this plugin implemented a node and a command adding the node to scene, connecting the node to the selected objects.
after install this plug-in, select an mesh object, then click the button "test material node", then
a node will be created and connected to the mesh object.

how to install in windows:
1.copy testMaterialNodeCreateUI.mel into ..\My Documents\maya\2011\scripts
2.install the plug-in:
  1>.window->Settings/Preferences->Plug-in Manager
  2>.click "Browse" on the left of the bottom, then go to where the .mll file is
  3>.then click "load"
  4>.^o^
